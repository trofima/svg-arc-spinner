Svg-arc-spinner usage example

Renders an svg arc spinner

View on GitHub Download .zip Download .tar.gz

Code Example

Include

<link rel="stylesheet" type="text/css" href="svg-arc-spinner/dist/SVGArcSpinner.min.css">

<script type="text/javascript" src="svg-arc-spinner/dist/SVGArcSpinner.min.js"></script>

Use

var spinnerContainer = document.getElementById('spinnerContainer');

spinnerContainer.innerHTML = new SVGArcSpinner({size: 200}).render();
